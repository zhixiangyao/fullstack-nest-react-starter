## Installation

- Prisma

```bash
# if it is running for the first time, pls run
pnpm api-service:db:init
# generator seed
pnpm api-service:db:seed
```

## Running the app

```bash
# watch mode
pnpm run api-service:start:dev

# production mode
pnpm run api-service:start:prod
```
